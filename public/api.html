Facebook Ads Automator API

API для автоматизации работы с Facebook Ads

GET/api/campaigns

Получение списка рекламных кампаний Facebook.

Пример ответа:

{
  "success": true,
  "message": "Fetched 25 campaigns.",
  "data": [
    {
      "name": "Campaign Name",
      "status": "ACTIVE",
      "id": "123456789"
    }
  ]
}

POST/api/sync-audience

Синхронизация аудитории с телефонными номерами.

Пример ответа:

{
  "success": true,
  "message": "Successfully synced audience with 157 phone numbers"
}

POST/api/create-lookalike

Создание Look-alike аудитории на основе существующей аудитории.

Пример ответа:

{
  "success": true,
  "message": "Successfully created Lookalike audience",
  "audienceId": "123456789"
}

GET/api/audiences

Получение списка всех Custom Audiences.

Пример ответа:

{
  "success": true,
  "message": "Fetched 5 audiences.",
  "data": [
    {
      "id": "123456789",
      "name": "Custom Audience Name",
      "type": "CUSTOM",
      "approximate_count": 1000
    }
  ]
}
